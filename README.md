# 🎓 Student Grade Manager (Java Console App)

## 📌 Description

A simple Java console-based application to manage student grades.
Users can input student details and the program calculates total, average, and grade.

---

## 🚀 Features

* Add multiple students
* Input marks for 3 subjects
* Calculate:

  * Total marks
  * Average marks
  * Grade (A / B / C)
* Display all student details

---

## 🛠️ Technologies Used

* Java
* OOP Concepts (Class, Object, Constructor, Methods)
* ArrayList
* Scanner (User Input)

---

## ▶️ How to Run

1. Compile the program:
   javac StudentManager.java

2. Run the program:
   java StudentManager

---

## 📊 Example Output

Enter number of students: 1
Enter name: Divya
Enter marks: 80 70 90

Output:
Name: Divya
Total: 240
Average: 80.0
Grade: A
